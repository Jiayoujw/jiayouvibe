隐私政策

最后更新日：2026年5月31日

1. 信息收集

jiayouvibe（下称"本网站"）不主动收集个人身份信息。本网站使用 localStorage 在您的浏览器中存储偏好（如主题选择、语言设置、收藏项目、笔记内容），这些数据仅存在于您的设备上，不会上传至任何服务器。

2. Cookie 和使用

本网站使用 Google AdSense 展示广告。Google 可能使用 Cookie 来展示个性化广告。详情请参阅 Google 的广告隐私政策。

3. GitHub API

本网站的"GitHub热门"功能通过 GitHub API 获取开源项目数据。API 请求包含用户认证令牌以提高请求额度，不传输任何用户个人信息。

4. 第三方服务

本网站可能包含第三方链接（如 GitHub、OpenAI、Google 等）。我们不对第三方网站的隐私政策负责。

5. 儿童隐私

本网站不面向13岁以下儿童。如发现收集了儿童信息，请联系我们删除。

6. 联系我们

如有隐私相关问题，请通过 jiayouvibe.com 的联系方式联系我们。
